# 普通心理学题库（AI可读整理版）

> 说明：本文件根据仓库中的 `题上.pdf`、`题下.pdf`、`记忆测验题目.pdf` 提取并整理而成，目的是让 AI 和人都能更快定位题型、主题和复习重点。

## 一、题库文件总览

### 1. `题库/普通心理学题/题上.pdf`
- 类型：单项选择题为主
- 覆盖范围：普通心理学综合内容
- 当前可见内容特征：
  - 大量单选题
  - 跨章节混合
  - 涉及心理学定义、神经基础、感觉知觉、记忆、思维、想象、注意、情绪、意志、能力、人格、意识等

### 2. `题库/普通心理学题/题下.pdf`
- 类型：多选题 + 简答题
- 覆盖范围：普通心理学综合内容
- 当前可见内容特征：
  - 多选题集中在中后段
  - 简答题适合背诵和问答训练
  - 有一定标准答案，可直接用于抽题与核对

### 3. `题库/普通心理学题/记忆测验题目.pdf`
- 类型：记忆专题题库
- 覆盖范围：第六章《记忆》
- 题型：单选、多选、名词解释、简答
- 适合用途：
  - 心理学先生按章节抽题
  - 记忆章节专项复习
  - 识别薄弱概念（如前瞻记忆、系列位置效应、组块、长时程增强等）

## 二、按题型分类

## 1. 单项选择题
主要来源：`题上.pdf`、`记忆测验题目.pdf`

### 代表主题
- 心理学研究对象与基本概念
- 神经系统与脑机制
- 感觉与知觉
- 注意
- 记忆
- 思维与想象
- 情绪情感
- 能力
- 人格与气质性格
- 意识
- 学习与问题解决

### 已识别的典型题目示例
- 艾宾浩斯发现的遗忘规律：遗忘进程是 **先快后慢**
- 心理学是一门研究 **心理现象发生、发展及活动规律** 的科学
- 听觉中枢位于 **颞叶**
- 短时记忆中把孤立项目结合成有意义单位叫 **组块**
- 瞬时记忆保持时间约 **0.25~2 秒**
- 头脑中呈现老师做实验的样子，属于 **情景记忆**
- 人的心理产生的源泉是 **客观现实**
- 与抽象思维关系密切的因素是 **语言**

### 记忆专题单选（来自 `记忆测验题目.pdf`）
- Sperling 局部报告法揭示：**感觉记忆** 的存在及容量
- 无法用语言清楚表达的记忆：**程序性记忆**
- 拼音干扰英语：**前摄抑制**
- 回到家乡触景生情：**情境依存性**

## 2. 多项选择题
主要来源：`题下.pdf`、`记忆测验题目.pdf`

### 代表主题
- 记忆分类
- 感觉与知觉特征
- 思维与概念
- 意识与人格
- 想象与幻想
- 气质与性格
- 意志品质

### 已识别的典型题目示例
- 倒摄抑制：
  - 表明遗忘可能是 **干扰的结果**
  - 指 **后学习材料** 对 **先学习材料回忆** 的干扰
- 错觉：
  - 是 **对客观事物的歪曲知觉**
  - **种类很多**
- 幻想：
  - 与 **生活愿望有关**
  - **指向未来**
  - 属于 **创造想象**
- 意识：
  - 是一种 **高级心理机能**
  - 具有 **统合、管理和调控作用**
  - 可以分为 **不同层次/水平**

### 记忆专题多选（来自 `记忆测验题目.pdf`）
- 图尔文将陈述性记忆分为：
  - **情景记忆**
  - **语义记忆**
- 短时记忆的信息编码方式有：
  - **听觉编码**
  - **语义编码**
  - **视觉编码**
- 按意识参与程度，可将记忆分为：
  - **外显记忆**
  - **内隐记忆**

## 五、给 AI 的使用说明（简版）

如果后续用于 AI 抽题/问答，推荐遵循：

1. 优先使用 `记忆测验题目.pdf` 进行章节专项抽题
2. 使用 `题上.pdf` 做随机单选训练
3. 使用 `题下.pdf` 做多选与简答训练
4. 名词解释和简答题优先回查 `专业书/` 原话
5. 用户答错时，不直接跳题；先讲解，再扩展追问 1~2 题
